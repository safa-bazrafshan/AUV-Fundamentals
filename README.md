# AUV Fundamentals

This repository contains all MATLAB scripts, slides, and resources used in the **AUV Fundamentals** YouTube playlist.  
The series introduces Autonomous Underwater Vehicles (AUVs) starting from basic concepts all the way to modeling, dynamics, navigation, and control implementation in MATLAB.
